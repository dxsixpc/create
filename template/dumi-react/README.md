# dumi-react

## 简介

这是基于 dumi 官方脚手架"create-dumi"的"React Library"模板生成的一套项目代码。

这里主要替换了一些项目配置，例如 eslint，prettier 等配置使用@dxsixpc/configs 里的通用配置

## 使用

使用可参考 dumi 官方文档https://d.umijs.org/

## 开发

```bash
# 安装依赖
$ npm install

# 本地调试运行文档
$ npm start

# 打包src下的资源代码
$ npm run build

# 动态打包src下的资源代码
$ npm run build:watch

# 打包文档
$ npm run docs:build

# 检查项目是否存在潜在问题
$ npm run doctor

# 提交代码
$ npm run commit
```
